## 趋势跟踪策略

趋势跟踪策略基于"趋势是你的朋友"理念，在趋势形成初期入场，趋势结束时离场。

### 1. MA交叉策略 (MAStrategy)

**策略原理**

利用两条不同周期的移动平均线交叉判断趋势方向。当快速MA上穿慢速MA时产生买入信号（金叉），下穿时产生卖出信号（死叉）。

**信号逻辑**

```
买入条件 (金叉):
  - 快MA[1] ≤ 慢MA[1]  (上一根K线快线在慢线下方)
  - 快MA[0] > 慢MA[0]  (当前K线快线穿越到慢线上方)

卖出条件 (死叉):
  - 快MA[1] ≥ 慢MA[1]  (上一根K线快线在慢线上方)
  - 快MA[0] < 慢MA[0]  (当前K线快线穿越到慢线下方)
```

**参数配置**

| 参数 | 默认值 | 说明 | 优化范围 |
|------|--------|------|----------|
| MA_FastPeriod | 10 | 快速MA周期 | 5-20 |
| MA_SlowPeriod | 20 | 慢速MA周期 | 20-60 |
| MA_Method | 0 | MA类型(0=SMA,1=EMA,2=SMMA,3=LWMA) | 0-3 |
| MA_SL_Pips | 30 | 止损点数 | 20-50 |
| MA_TP_Pips | 60 | 止盈点数 | 40-100 |

**适用场景**

- 时间框架: H1、H4
- 市场状态: 明显趋势行情
- 品种: EURUSD、GBPUSD、USDJPY

**优缺点**

| 优点 | 缺点 |
|------|------|
| 逻辑简单，易于理解 | 震荡市容易假信号 |
| 趋势行情获利能力强 | 信号滞后 |
| 参数可调性强 | 需要配合止损使用 |

### 2. MACD策略 (MACDStrategy)

**策略原理**

MACD（Moving Average Convergence Divergence）通过计算两条指数移动平均线的差值来判断趋势动能。策略在MACD线与信号线交叉时产生交易信号，并使用零轴过滤提高信号质量。

**信号逻辑**

```
买入条件:
  - MACD主线[1] ≤ 信号线[1]
  - MACD主线[0] > 信号线[0]  (上穿)
  - MACD主线[0] < 阈值       (在零轴下方，确保不是高位追多)

卖出条件:
  - MACD主线[1] ≥ 信号线[1]
  - MACD主线[0] < 信号线[0]  (下穿)
  - MACD主线[0] > -阈值      (在零轴上方)
```

**参数配置**

| 参数 | 默认值 | 说明 | 优化范围 |
|------|--------|------|----------|
| MACD_Fast | 12 | 快线周期 | 8-15 |
| MACD_Slow | 26 | 慢线周期 | 20-30 |
| MACD_Signal | 9 | 信号线周期 | 7-12 |
| MACD_SignalThreshold | 0 | 零轴过滤阈值 | 0-0.001 |
| MACD_SL_Pips | 40 | 止损点数 | 30-60 |
| MACD_TP_Pips | 80 | 止盈点数 | 60-120 |

**适用场景**

- 时间框架: H1、H4、D1
- 市场状态: 趋势行情，动能转换点
- 品种: 所有主要货币对

**MACD柱状图解读**

```
        零轴
    ──────────────
    ▓▓▓▓▓▓▓        ← 正值柱（多头动能）
    ▓▓▓▓▓▓▓▓▓
    ▓▓▓▓▓▓▓▓▓▓▓▓
    ──────────────
       ▓▓▓▓▓       ← 负值柱（空头动能）
       ▓▓▓▓▓▓▓
       ▓▓▓▓▓▓▓▓

    柱体增长 = 动能增强
    柱体缩短 = 动能减弱
```

## 均值回归策略

均值回归策略基于"价格会回归均值"的统计特性，在价格偏离均值过远时反向入场。

### 3. RSI均值回归策略 (RSIStrategy)

**策略原理**

RSI（Relative Strength Index）衡量价格变动的速度和幅度。当RSI进入超买区（>70）或超卖区（<30）时，价格可能即将反转。策略在RSI从极值区回归时入场。

**信号逻辑**

```
买入条件 (确认模式):
  - RSI[1] ≤ 超卖阈值(30)     (上一根K线在超卖区)
  - RSI[0] > 超卖阈值         (当前K线离开超卖区)
  - RSI[0] > RSI[上一值]      (RSI开始上升)

卖出条件 (确认模式):
  - RSI[1] ≥ 超买阈值(70)     (上一根K线在超买区)
  - RSI[0] < 超买阈值         (当前K线离开超买区)
  - RSI[0] < RSI[上一值]      (RSI开始下降)
```

**参数配置**

| 参数 | 默认值 | 说明 | 优化范围 |
|------|--------|------|----------|
| RSI_Period | 14 | RSI计算周期 | 7-21 |
| RSI_Overbought | 70 | 超买阈值 | 65-80 |
| RSI_Oversold | 30 | 超卖阈值 | 20-35 |
| RSI_UseConfirmation | true | 使用确认模式 | true/false |
| RSI_SL_Pips | 25 | 止损点数 | 15-40 |
| RSI_TP_Pips | 50 | 止盈点数 | 30-80 |

**RSI区域划分**

```
    100 ┤
        │  极度超买区
     80 ┤────────────────
        │  超买区 (考虑卖出)
     70 ┤────────────────
        │
        │  中性区
        │
     30 ┤────────────────
        │  超卖区 (考虑买入)
     20 ┤────────────────
        │  极度超卖区
      0 ┤
```

**适用场景**

- 时间框架: H1、H4
- 市场状态: 震荡行情、区间交易
- 品种: EURUSD、USDJPY（波动相对规律）

### 4. 布林带策略 (BollingerStrategy)

**策略原理**

布林带由中轨（移动平均线）和上下轨（中轨±N倍标准差）组成。价格触及上轨后回落时做空，触及下轨后反弹时做多。策略假设价格会回归中轨。

**信号逻辑**

```
买入条件:
  - 收盘价[1] ≤ 下轨[1]      (上一根K线触及或穿透下轨)
  - 收盘价[0] > 下轨[0]      (当前K线反弹离开下轨)

卖出条件:
  - 收盘价[1] ≥ 上轨[1]      (上一根K线触及或穿透上轨)
  - 收盘价[0] < 上轨[0]      (当前K线回落离开上轨)

止盈设置 (回归中轨模式):
  - 止盈价格 = 布林带中轨
```

**参数配置**

| 参数 | 默认值 | 说明 | 优化范围 |
|------|--------|------|----------|
| BB_Period | 20 | 布林带周期 | 15-25 |
| BB_Deviation | 2.0 | 标准差倍数 | 1.5-2.5 |
| BB_ReturnToMean | true | 回归中轨止盈 | true/false |
| BB_SL_Pips | 20 | 止损点数 | 15-35 |
| BB_TP_Pips | 40 | 固定止盈点数 | 25-60 |

**布林带形态解读**

```
        上轨 (中轨 + 2σ)
    ────────────────────────
    ╱                      ╲
   ╱                        ╲
──┼────────── 中轨 ──────────┼──
   ╲                        ╱
    ╲                      ╱
    ────────────────────────
        下轨 (中轨 - 2σ)

    带宽收窄 = 波动率降低，可能突破
    带宽扩张 = 波动率增加，趋势确立
```

**适用场景**

- 时间框架: H1、H4
- 市场状态: 震荡行情
- 品种: 所有货币对

## 突破策略

突破策略在价格突破关键水平时入场，捕捉趋势启动点。

### 5. 通道突破策略 (ChannelBreakout)

**策略原理**

基于唐奇安通道（Donchian Channel）原理，在指定周期内寻找最高价和最低价形成通道。当价格突破通道上沿时做多，突破下沿时做空。

**信号逻辑**

```
通道计算:
  - 通道高点 = 过去N根K线的最高价
  - 通道低点 = 过去N根K线的最低价
  (计算时跳过当前K线，避免未来数据)

买入条件:
  - 收盘价 > 通道高点 + 入场偏移

卖出条件:
  - 收盘价 < 通道低点 - 入场偏移

冷却机制:
  - 每次突破后60分钟内不再产生新信号
```

**参数配置**

| 参数 | 默认值 | 说明 | 优化范围 |
|------|--------|------|----------|
| CB_ChannelPeriod | 20 | 通道周期 | 15-30 |
| CB_EntryOffset | 0 | 入场偏移点数 | 0-10 |
| CB_SL_Pips | 30 | 固定止损点数 | 20-50 |
| CB_TP_Pips | 60 | 止盈点数 | 40-100 |
| CB_UseATR_SL | false | 使用ATR动态止损 | true/false |
| CB_ATR_Period | 14 | ATR周期 | 10-20 |
| CB_ATR_Multiplier | 2.0 | ATR止损倍数 | 1.5-3.0 |

**ATR动态止损**

```
ATR止损 = 入场价 ± (ATR × 倍数)

示例:
  - ATR = 0.0020 (20点)
  - 倍数 = 2.0
  - 止损距离 = 40点
```

**适用场景**

- 时间框架: H1、H4
- 市场状态: 突破行情、趋势启动
- 品种: XAUUSD、GBPUSD（波动大）

### 6. 支撑阻力突破策略 (SupportResistance)

**策略原理**

自动识别历史支撑和阻力位，当价格突破这些关键水平时入场。策略要求支撑/阻力位被多次触及确认其有效性。

**信号逻辑**

```
支撑阻力识别:
  1. 在回溯周期内找到最高价和最低价
  2. 最高价 = 阻力位
  3. 最低价 = 支撑位

有效性确认:
  - 支撑/阻力位被触及次数 ≥ 设定值(默认2次)

买入条件:
  - 阻力位有效(触及次数足够)
  - 收盘价[1] < 阻力位
  - 收盘价[0] > 阻力位 (向上突破)

卖出条件:
  - 支撑位有效(触及次数足够)
  - 收盘价[1] > 支撑位
  - 收盘价[0] < 支撑位 (向下突破)
```

**参数配置**

| 参数 | 默认值 | 说明 | 优化范围 |
|------|--------|------|----------|
| SR_Lookback | 50 | 回溯K线数 | 30-100 |
| SR_TouchCount | 2 | 最小触及次数 | 2-5 |
| SR_ProximityPips | 10 | 近距离判断点数 | 5-15 |
| SR_SL_Pips | 25 | 止损点数 | 20-40 |
| SR_TP_Pips | 50 | 止盈点数 | 40-80 |

**支撑阻力示意图**

```
价格
  ↑
  │         ╱╲      ╱╲
  │        ╱  ╲    ╱  ╲     阻力位 R
  │    ╱╲ ╱    ╲  ╱    ╲ ────────────
  │   ╱  ╲      ╲╱      ╲
  │  ╱    ╲    ╱╲       ╱╲
  │ ╱      ╲  ╱  ╲     ╱  ╲
  │╱        ╲╱    ╲   ╱    ╲ ──────────── 支撑位 S
  │                ╲ ╱
  └──────────────────────────────→ 时间

  突破阻力位 R → 做多
  跌破支撑位 S → 做空
```

**适用场景**

- 时间框架: H4、D1
- 市场状态: 区间突破
- 品种: 所有品种

## 剥头皮策略

### 7. 快速剥头皮策略 (QuickScalp)

**策略原理**

剥头皮策略追求小止损、快止盈，通过高频交易积累利润。策略结合快速MA交叉和RSI过滤，只在趋势方向确认时入场。

**信号逻辑**

```
入场条件:
  买入: 快MA上穿慢MA + RSI > 50 (多头确认)
  卖出: 快MA下穿慢MA + RSI < 50 (空头确认)

过滤条件:
  - 点差检查: 当前点差 ≤ 最大允许点差
  - 交易间隔: 距上次交易 ≥ 最小间隔(默认30秒)
```

**参数配置**

| 参数 | 默认值 | 说明 | 优化范围 |
|------|--------|------|----------|
| QS_FastMA | 5 | 快速EMA周期 | 3-8 |
| QS_SlowMA | 13 | 慢速EMA周期 | 10-15 |
| QS_RSI_Period | 7 | RSI周期 | 5-10 |
| QS_RSI_Level | 50 | RSI过滤水平 | 45-55 |
| QS_SL_Pips | 10 | 止损点数 | 8-15 |
| QS_TP_Pips | 15 | 止盈点数 | 10-25 |
| QS_MaxSpread | 3 | 最大允许点差 | 1-5 |
| QS_FilterSpread | true | 过滤高点差 | true/false |

**风险特性**

```
盈亏比: 1:1.5 (止损10点/止盈15点)

胜率要求:
  - 盈亏比1:1.5时，胜率 > 40% 即可盈利
  - 目标胜率: 55-65%

日交易次数: 5-15次
```

**适用场景**

- 时间框架: M1、M5
- 市场状态: 流动性充足时段
- 品种: EURUSD、USDJPY（点差小）
- 时段: 伦敦时段、纽约时段

**剥头皮注意事项**

1. 选择低点差经纪商
2. 确保执行速度快
3. 避免重大新闻前后交易
4. 严格执行止损

## 风险管理规则

### 仓位计算

```
仓位 = (账户余额 × 风险百分比) / (止损点数 × 每点价值)

示例:
  - 账户余额: $10,000
  - 风险百分比: 1.5%
  - 止损点数: 30点
  - 每点价值: $10/手 (EURUSD)

  仓位 = (10000 × 0.015) / (30 × 10) = 0.05手
```

### 风险限制

| 限制项 | 默认值 | 说明 |
|--------|--------|------|
| 单笔风险 | 1.5% | 每笔交易最大亏损占账户比例 |
| 日最大亏损 | 5.0% | 单日累计亏损上限 |
| 周最大亏损 | 10.0% | 单周累计亏损上限 |
| 最大回撤 | 15.0% | 触发后暂停交易 |
| 日最大交易次数 | 5次 | 防止过度交易 |
| 最大持仓数 | 2个 | 分散风险 |

### 移动止损

```
触发条件: 盈利 ≥ 移动止损启动点数

做多: 新止损 = 当前价 - 移动止损步长
做空: 新止损 = 当前价 + 移动止损步长

只向盈利方向移动，不回撤
```

## 参数优化指南

### 回测流程

1. **选择回测周期**: 至少6个月历史数据
2. **设置点差**: 使用实际点差或+10%缓冲
3. **模型选择**: "每个tick"最准确
4. **优化指标**:
   - 净利润
   - 最大回撤
   - 盈利因子
   - 夏普比率

### 参数优化原则

```
过度拟合检测:
  - 样本内优化 → 样本外验证
  - 参数敏感性分析
  - Walk-forward分析

稳健参数特征:
  - 参数小幅变化不影响整体表现
  - 多个品种表现稳定
  - 样本外测试盈利
```

### 各品种推荐参数

| 品种 | 策略 | 时间框架 | 主要参数调整 |
|------|------|----------|--------------|
| EURUSD | MA交叉 | H1 | 快MA=10, 慢MA=20, SL=30 |
| GBPUSD | MACD | H1 | SL=50, TP=100 (波动大) |
| USDJPY | RSI | H4 | 超买=70, 超卖=30 |
| XAUUSD | 通道突破 | H4 | 周期=30, SL=100 (波动大) |
| XAGUSD | 不建议剥头皮 | - | 波动过大 |
